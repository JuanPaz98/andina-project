Primeramente muchas gracias por haberme brindado esta grandiosa oportunidad y ponerme la prueba de hacer una SPA en el lado del Front con un framework como Angular, el cual desconocía totalmente. No pude culminar la prueba por cuestión de tiempo y porque tuve que estudiar varias horas desde cero el framework para entender un poco la lógica e iniciar con la creación de la aplicación, sin embargo le dediqué todo mi tiempo y esfuerzo. 

Angular me pareció una muy buena tecnología en general, es muy robusto y abarca todas las necesidades en un mismo stack de herramientas o librerías. Con esta prueba aprendí sobre las bases de Angular, aprendí a generar módulos y componentes, aprendí como manejar rutas, conocí Angular Material (que no me gustó mucho) y detallé lo bien estructurado que está todo por defecto, a diferencia de React donde uno mismo crea sus componentes de manera manual y no hay un orden preestablecido.

De la prueba alcancé a cumplir lo siguiente:

- Hice las vistas y las maqueté con css puro.

- Generé los componentes de creación de producto, lista de productos, un navbar para navegar entre vistas y un componente de login. 

- Generé un "backend" con Json server, el cual me servía como base de datos para hacer consulta de productos. 

- Hice las rutas principales de login, productos y creación de productos. 


No alcancé a hacer lo siguiente:

- CRUD de productos, ya que no pude hacer los métodos POST, UPDATE Y DELETE desde el modal (que tampoco alcancé a maquetar)

- No pude relacionar los productos con las ciudades como se solicitó en la prueba. 

- Por cuestiones de tiempo no pude generar una data para registro y login de usuarios ni tampoco pude generar sesiones con autenticación.
